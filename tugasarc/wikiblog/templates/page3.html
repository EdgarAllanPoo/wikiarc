Cara Mengetahui IP Address Komputer Lain Dalam Satu Jaringan

Mengetahui cara memindai jaringan untuk perangkat adalah langkah pertama, dan salah satu yang paling mendasar, dalam mengelola IP address. Jika kalian menggunakan sistem operasi Windows. Ikuti langkah berikut untuk mengetahui IP address komputer lain.

Cara Mengetahui IP Address Komputer Lain Di Windows

  1. Pertama hubungkan PC/Laptop kalian ke jaringan LAN atau Wifi, kemudian tekan simbol Windows + R lalu ketikan CMD.
  2. Selanjutnya ketikkan perintah “arp -a -v” tanpa tanda petik di Command Prompt lalu tekan Enter.
  3. Dengan perintah tersebut, maka nanti akan muncul daftar IP address yang terhubung di satu jaringan. Kalian dapat melihat MAC Address, kalian juga akan dapat mengetahui IP Address Komputer, IP Address Modem, dan IP Address device yang lain namun dengan satu syarat semua perangkat komputer tersebut harus terhubung dalam satu jaringan yang sama.

Cara Mengetahui IP Address Komputer Lain Di Linux

Cara paling dasar untuk menemukan semua IP address di jaringan adalah dengan pemindaian jaringan manual. Metode ini paling baik bagi mereka yang ingin melakukan pemeriksaan perangkat dengan lebih cepat. Jika kalian menggunakan sistem operasi Linux sebanarnya ada beberapa cara untuk melakukan pemindaian IP address, tetapi adamin akan membagikan satu cara yang umum digunakan. Untuk memindai sendiri jaringan dengan cepat menggunakan sistem operasi Linux, ikuti langkah berikut ini.

  1. Pastikan PC/Laptop kalian terhubung ke jaringan, kemudian buka terminal Linux kalian.
  2. Setelah terbuka, ketikan perintah “sudo arp-scan -l” di terminal linux kalian.
  3. Dengan perintah tersebut, kalian akan memindai dengan cepat semua perangkat yang aktif dan mengetahui IP address, MAC Address, dan merek dari perangkat yang terhubung ke dalam satu jaringan yang sama.
