Cara Ping Koneksi Internet dengan CMD

Ping adalah singkatan dari Packet Internet Gropper, sebuah perintah yang digunakan untuk mengecek sebuah koneksi jaringan yang biasanya dipakai melalui pinh command promprt atau biasa disingkat CMD di PC.

Ping

  1. Pertama tekan logo Windows + R pada keyboard PC secara bersamaan
  2. Lalu masukkan perintah Ping berikut ini pada kolom CMD

    3. ping www.google.com -t

  3. Lalu tekan Enter atau Oke pada menu CMD
  4. Tunggu beberapa saat jika koneksi internet stabil maka ping di CMD akan segera muncul
  5. Semakin kecil nilai time= pada ping semakin bagus koneksi internet kami, jika jaringan terjadi error maka akan menandakan pesan request timed out.
